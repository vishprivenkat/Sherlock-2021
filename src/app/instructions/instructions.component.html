Instructions



Before Playing Sherlock

  1. Sign up at kurukshetraceg.org.in.

  2. Get your K! ID

  3. This is a multiplayer game.

  4. Make sure your partner, Watson has signed up at kurukshetraceg.org.in

  5. Have Watson's K! ID handy too.

  6. Email ID and Password used for logging in Sherlock site should be same as the one used for signup in kurukshetraceg.org.in

  7. If Google Login was used in signup in kurukshetraceg.org.in, the same account should be used for logging in Sherlock Game Site.



Pairing Sherlock and Watson

  1. Sherlock should login in the Website

  2. Sherlock will be taken to the pair up page.

  3. Sherlock is required to enter Watson's K! ID as per instructions stated there.

  4. Watson will recieve an email containing a link.

  5. Watson should click the link in the email to pair up with Sherlock within 24 Hrs of sending the pairing request.

  6. Only if paired, can Sherlock and Watson proceed to play the game.



Playing the Game

  1. Game will proceed level wise.

  2. Sherlock and Watson should play together.

  3. Only if both players complete the current level, can they proceed to the next level.

  4. Answer should be entered in the format stated there in the answer box.

  5. There are finite number of tries for answering the questions. If player exhausts the number of tries, then contact for assisstance to proceed.

  6. Clues are available for all levels

  7. Some levels have audio clues and player will be prompted regarding the same.

  8. Players can check their rankings in the leaderboard



Read all the slides carefully and then answer. Only then, you can understand the storyline properly.
